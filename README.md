# 🌴 Greens Malayalee Kootayama (GMK) Community Platform

> **"Kootayama"** represents collectiveness, unity, and shared cultural bonds in Malayalam. 

The **GMK Community Platform** is a secure, modern, mobile-responsive digital ecosystem tailored for a gated residential community of over 100+ families. This application streamlines resident verification, profiles, financial ledger updates, and event coordination for major yearly cultural events like **Onam** and **GMAD (Greens Malayalee Annual Day)**.

This repository features an interactive, high-fidelity deployment prototype built on utility-first presentation engines and light client-side state management frameworks.

---

## 🏗️ Core System Architecture & Features

The platform handles complex data integrity requirements across three distinct functional modules:

### 1. Advanced Resident Intake Lifecycle
* **Dynamic Username Blueprint:** The system strips spaces and automatically maps structural addresses to sanitized, zero-padded uppercase primary system usernames (e.g., `B3-04-72` transforms instantly to `B30472`).
* **Multi-Member Data Storage:** Captures interconnected structural variables for households, including professional profiles, employment dependencies, and age brackets for children.
* **Cryptographic OTP Handshake:** Features simulated infrastructure tokenization gateways to guarantee phone profile verification.

### 2. Multi-Variant Dynamic Pricing Engine
* Evaluates active family member configurations and calculates precise pricing tiers automatically using local currency variables (**OMR**).
* Applies accurate adjustments for spouse employment types, child age brackets, and external guest counts on the fly.

### 3. High-Security Administrative Control Center
* **Live Activity Logging Router:** Tracks security events, database updates, and authentication handshakes in real time.
* **Dual QR Checkpoint Subsystem:** Simulates independent data tracking structures for main entrance attendance records and dining hall food voucher collection logs to prevent duplicate entry attempts.
* **Consolidated Financial Ledger:** Provides accounting tracking for community ticket sales and corporate sponsorship injections to deliver instant net liquidity values.

## 🛠️ Built-With Technology Stack

This single-file deployment prototype leverages high-density modern frontend libraries:
* **Presentation Layer:** Tailwind CSS (configured with a custom premium Kerala-inspired emerald and gold cultural palette).
* **State Management & Routing:** Alpine.js (reactive state engine driving the multi-step forms and dashboards without backend page refreshes).
* **Typography & Design:** Inter & Cinzel serif typefaces for an elegant, high-end design aesthetic.

## 🚀 Instant Deployment Guide (GitHub Pages)

This single-file solution is optimized for zero-overhead serverless hosting using **GitHub Pages**.

1. **Clone or Download** this repository.
2. Ensure your primary file is named exactly **`index.html`** and sits in the root directory.
3. Push your repository to **GitHub** and ensure its visibility status is set to **Public**.
4. Navigate to your repository's **Settings** ➡️ **Pages**.
5. Under **Build and deployment**, drop down the branch selector and switch it from `None` to **`main`** (or `master`), then click **Save**.
6. Refresh the interface after 60 seconds to access your unique, permanent cloud application URL.
